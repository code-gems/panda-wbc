# Welcome to Panda Component
Panda Full Page

### Installation
```npm install @panda-wbc/panda-slideshow -S```

or 

```yarn add @panda-wbc/panda-slideshow```

### Usage

```html
<panda-slideshow
	
>
	<panda-slideshow-slide>
		Page 1
	</panda-slideshow-slide>
	<panda-slideshow-slide>
		Page 2
	</panda-slideshow-slide>
	<panda-slideshow-slide>
		Page 3
	</panda-slideshow-slide>
</panda-slideshow>
```
