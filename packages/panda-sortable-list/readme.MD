# Welcome to Panda Sortable List


### Installation
```bash
npm install @panda-wbc/panda-sortable-list -S
```

or 

```bash
yarn add @panda-wbc/panda-sortable-list
```

### Usage

```html
<panda-sortable-list
>
</panda-sortable-list>
```
