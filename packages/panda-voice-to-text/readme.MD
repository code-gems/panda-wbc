# Welcome to Panda Voice to Text


### Installation
```bash
npm install @panda-wbc/panda-voice-to-text -S
```

or 

```bash
yarn add @panda-wbc/panda-voice-to-text
```

### Usage

```html
<panda-voice-to-text></panda-voice-to-text>
```
