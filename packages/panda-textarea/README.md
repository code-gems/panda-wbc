# Welcome to Panda Component
Panda Text Area

### Installation
```npm install @panda-wbc/panda-textarea```

or 

```yarn add @panda-wbc/panda-textarea```

### Usage

```html
<panda-textarea
	maxlength="256"
	show-length
	resizable
	autoselect
	rows="4"
	@change="${_handleChange}"
>
</panda-textarea>
```
