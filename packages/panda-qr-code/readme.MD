# Welcome to Panda QR Code


### Installation
```bash
npm install @panda-wbc/panda-qr-code -S
```

or 

```bash
yarn add @panda-wbc/panda-qr-code
```

### Usage

```html
<panda-qr-code></panda-qr-code>
```
