# Welcome to Panda Theme
Panda Theme is a Lit based web component that introduces few themes for panda web components.

### Installation
```npm install @panda/panda-theme -S```

or 

```yarm add @panda/panda-theme```

### Usage

```html
<panda-theme
	.theme="light"
></panda-theme>
```
