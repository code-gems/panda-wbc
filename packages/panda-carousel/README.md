# Welcome to Panda Component
Panda Carousel

### Installation
```npm install @panda-wbc/panda-carousel```

or 

```yarn add @panda-wbc/panda-carousel```

### Usage

```html
<panda-carousel
	
>
	<panda-carousel-slide>
		Slide 1
	</panda-carousel-slide>
	<panda-carousel-slide>
		Slide 2
	</panda-carousel-slide>
	<panda-carousel-slide>
		Slide 3
	</panda-carousel-slide>
</panda-carousel>
```
