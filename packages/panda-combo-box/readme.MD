# Welcome to Panda Combo Box


### Installation
```bash
npm install @panda-wbc/panda-combo-box -S
```

or 

```bash
yarn add @panda-wbc/panda-combo-box
```

### Usage

```html
<panda-combo-box
	.value="${...}"
	.items="${...}"
	.disabled="${true/false}"
	.busy="${true/false}"
	@change="${...}"
>
</panda-combo-box>
```
