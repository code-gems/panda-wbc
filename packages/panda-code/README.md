# Welcome to Panda Code

### Installation
```npm install @panda-wbc/panda-code```

or 

```yarn add @panda-wbc/panda-code```

### Usage

```html
<panda-code lang="html">
	const sum = (a: number, b: number): number => {
		return a + b;
	}

	<button onclick="alert(sum(2, 2));">
		This Sums it Up
	</button>
</panda-code>
```
