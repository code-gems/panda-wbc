# Welcome to Panda Chart
Panda Chart is a Lit based web component for data visualization. Extremely lightweight and easy to use.
Enjoy!

### Installation
```npm install @panda/panda-chart -S```

or 

```yarm add @panda/panda-chart```

### Usage

```html
<panda-chart
	.dataSet="${...}"
></panda-chart>
```
