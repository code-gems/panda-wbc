# Welcome to Panda Checkbox Group
Panda Checkbox is a native web component. It compliments checkbox component.
Enjoy!

### Installation
```npm install @panda-wbc/panda-checkbox-group```

or 

```yarn add @panda-wbc/panda-checkbox-group```

### Usage

```html
<panda-checkbox-group label="Select your gender:">
	<panda-checkbox name="gender">Male</panda-checkbox>
	<panda-checkbox name="gender">Female</panda-checkbox>
</panda-checkbox-group>
```
