# Welcome to Panda Heatmap component
Panda Heatmap is a native vanilla javascript web component that allows you to display awesome heatmaps on your UI. 
Extremely lightweight and easy to use.
Enjoy!

### Installation
```npm install @panda-wbc/panda-heatmap```

or 

```yarn add @panda-wbc/panda-heatmap```

### Usage

```html
<panda-heatmap
	.data="${usersHeatmapData}"
></panda-heatmap>
```
