# Welcome to Panda Text Field


### Installation
```npm install @panda-wbc/panda-text-field```

or 

```yarn add @panda-wbc/panda-text-field```

### Usage

```html
<panda-text-field
	@on-input="${...}"
>
	<div slot="prefix">{prefix icon}</div>
	<div slot="suffix">{suffix icon}</div>
</panda-text-field>
```
