# Welcome to Panda Slider
Panda Slider is a Lit based web component. Extremely lightweight and easy to use.
Enjoy!

### Installation
```npm install @panda-wbc/panda-slider```

or 

```yarn add @panda-wbc/panda-slider```

### Usage

```html
<panda-slider
	theme="done"
	.min="${50}"
	.max="${200}"
	.value="${125}"
	.busy="${true|false}"
	.disabled="${true|false}"
></panda-slider>
```
