# Welcome to Panda Component
Panda Full Page

### Installation
```npm install @panda-wbc/panda-full-page```

or 

```yarn add @panda-wbc/panda-full-page```

### Usage

```html
<panda-full-page
	
>
	<panda-full-page-content>
		Page 1
	</panda-full-page-content>
	<panda-full-page-content>
		Page 2
	</panda-full-page-content>
	<panda-full-page-content>
		Page 3
	</panda-full-page-content>
</panda-full-page>
```
