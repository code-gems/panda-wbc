### Installation
```npm install @panda-wbc/panda-multi-select-combo-box```

or 

```yarn add @panda-wbc/panda-multi-select-combo-box```

### Usage

```html
<panda-multi-select-combo-box
	@change="${(event: PandaMultiSelectComboBoxChangeEvent) => onChange(event)}"
></panda-multi-select-combo-box>
```
